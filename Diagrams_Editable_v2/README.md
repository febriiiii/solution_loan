Berisi 12 diagram dalam 3 format supaya bisa diedit dengan tool GRATIS.

## Isi folder
- `mermaid/` — file `.mmd` (kode sumber diagram). **Paling mudah diedit.**

- `svg/`     — file `.svg` (gambar vektor; bisa diimpor ke draw.io / Inkscape).

## Cara edit (semua GRATIS, tanpa install)

### Opsi A — mermaid.live (REKOMENDASI, edit penuh)
1. Buka **https://mermaid.live**
2. Buka file `.mmd` dengan text editor, copy isinya
3. Paste di panel kiri → diagram langsung tampil & bisa diedit (ubah node, label, panah)
4. Export PNG/SVG lewat menu **Actions → PNG/SVG**

### Opsi B — diagrams.net dengan Mermaid native (edit sebagai shape asli)
1. Buka **https://app.diagrams.net**
2. Menu **Arrange → Insert → Advanced → Mermaid**
3. Paste isi file `.mmd` → diagram jadi **shape draw.io yang bisa digeser/diedit satuan**
4. Export lewat **File → Export as → PNG/SVG/PDF**

### Opsi C — buka file .drawio langsung
1. Buka **https://app.diagrams.net** → **Open Existing Diagram** → pilih file `.drawio`
2. Diagram tampil sebagai gambar vektor; bisa ditambah anotasi/kotak/teks di atasnya
3. Export sesuai kebutuhan

### Opsi D — file .svg
Impor ke **draw.io** atau **Inkscape** (gratis) untuk edit vektor per-elemen.

> Tip: untuk mengubah **isi** diagram (tambah tabel di ERD, ubah alur), pakai **Opsi A atau B** (.mmd). Untuk sekadar menambah catatan/anotasi di atas gambar, **Opsi C** (.drawio) paling cepat.
